# TriInterWrapperWettedPerimIC

!syntax description /ICs/TriInterWrapperWettedPerimIC

## Overview

<!-- -->

This kernel creates an initial field for the inter-wrapper wetted perimeter ([!param](/ICs/TriInterWrapperWettedPerimIC/variable) = `w_perim`), for an inter-wrapper that wraps around sub-assemblies with a triangular subchannel/pin arrangement.
