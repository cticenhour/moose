# QuadFlowAreaIC

!syntax description /ICs/QuadFlowAreaIC

## Overview

<!-- -->

This kernel populates the auxiliary variable S (subchannel surface area) at the beginning of the problem
for a sub-assembly with a square subchannel/pin arrangement.
